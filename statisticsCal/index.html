Statistics Calculator!

Write the wed app to prompt accepting for a positive integer until the user enters a negative integer

  • If the entered text is not a positive integer number, then the web app will keep asking for a number until the valid possitive integer is entered
  • If the entered text is a negative integer, then the web app will show the statistics (average,min,max) of all entered positive integers

Hint:

  • You need to implement function to check whether an entered text is a positive integer or not and whether an entered text is a negative integer or not
  • You need to implement functions to compute average,min and max of a list of integer
